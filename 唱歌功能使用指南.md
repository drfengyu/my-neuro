# 🎤 My-Neuro 唱歌功能使用指南

## 📋 功能概述

肥牛可以唱歌！使用本地下载的网易云音乐，配合人声/伴奏分离技术，实现：
- 🎵 播放歌曲并同步嘴型动画
- 🎤 自动触发麦克风握持动作
- 📝 实时显示歌词气泡
- 🔊 人声/伴奏分离播放（更真实的唱歌效果）

---

## 🚀 快速开始

### 1. 下载歌曲

使用内置的网易云音乐下载工具：

```bash
# 搜索歌曲
node download-music.js 告白气球

# 直接下载（使用歌曲ID）
node download-music.js 25906124

# 批量下载
node download-music.js 25906124 25906125 25906126
```

**下载位置**：`song-library/downloads/`

每首歌会生成两个文件：
- `歌名-歌手.mp3` - 完整音频
- `歌名-歌手.json` - 元数据（歌词、时长等）

---

### 2. 人声/伴奏分离（可选，推荐）

如果要实现更真实的唱歌效果，需要分离人声和伴奏。

**方法A：使用在线工具**
1. 访问 [vocal-remover.org](https://vocalremover.org/) 或 [lalal.ai](https://www.lalal.ai/)
2. 上传 `歌名-歌手.mp3`
3. 下载分离后的两个文件：
   - `歌名-歌手-Vocal.mp3` - 人声
   - `歌名-歌手-Acc.mp3` - 伴奏

**方法B：使用本地工具（需要GPU）**
```bash
# 安装 Spleeter 或 Demucs
pip install spleeter

# 分离音频
spleeter separate -p spleeter:2stems song-library/downloads/歌名-歌手.mp3 -o song-library/output
```

**输出位置**：将分离后的文件放到 `song-library/output/`

---

### 3. 播放歌曲

**方法A：快捷键（推荐）**
- 按 **Ctrl+Shift+6** 随机播放歌曲

**方法B：HTTP API**
```bash
# 访问控制端口
http://localhost:1001/music/play
```

**方法C：代码调用**
```javascript
global.musicPlayer.playRandomMusic();
```

## 🎯 播放模式

### 模式1：单音频播放
- 只有完整音频文件
- 嘴型同步整个音频的频谱
- 适合快速测试

### 模式2：双音频播放（推荐）
- 人声 + 伴奏分离
- 嘴型**只**同步人声（更真实）
- 伴奏作为背景音乐
- 人声音量可调节

## 🐛 常见问题

### Q1: 按 Ctrl+Shift+6 没反应？
**A**: 检查 `song-library/output/` 目录是否有歌曲文件

### Q2: 播放没有嘴型动画？
**A**: 确保音频文件格式正确，且浏览器支持该格式

### Q3: 想要更好的唱歌效果？
**A**: 使用人声/伴奏分离功能！这样嘴型只同步人声部分

### Q4: 如何添加更多歌曲？
**A**: 
1. 使用 `download-music.js` 下载
2. 分离人声和伴奏
3. 放到 `song-library/output/`

## 🎨 技术实现

### 嘴型同步原理
1. 使用 Web Audio API 分析音频频谱
2. 计算低频和高频能量
3. 根据能量值调节模型嘴巴开合度
4. 实时更新（60fps）

### 动作触发
- 播放开始：触发"握麦克风"动作（Motion 8）
- 播放结束：恢复默认动作
- 播放错误：恢复默认动作

### 歌词同步
- 从 `.json` 元数据读取歌词
- 每行歌词显示在气泡框中
- 根据歌曲时长自动计算显示时间

## 🔧 高级选项

### 调节人声音量
编辑 `music-player.js` 第103行：
```javascript
vocalGain.gain.value = 0.8; // 0.0-1.0，默认0.8
```

### 修改音乐文件夹
编辑 `music-player.js` 第9行：
```javascript
this.musicFolder = 'song-library\\output';
```
