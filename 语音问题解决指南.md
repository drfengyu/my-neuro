## 🔍 故障排查

### 如果重启后仍然 volcEnabled=false

1. **检查JSON格式**：
   ```bash
   node -e "JSON.parse(require('fs').readFileSync('live-2d/config.json','utf8'))"
   ```
   如果报错，说明JSON格式有问题

2. **手动编辑配置**：
   - 用记事本打开 `live-2d\config.json`
   - 找到 `"volcengine_tts"`
   - 确保 `"enabled": true` (注意逗号)
   - 找到 `"tts"`
   - 确保 `"enabled": false`

3. **检查是否有多个config.json**：
   ```bash
   find . -name "config.json" -type f
   ```
   程序可能在读取错误的文件
