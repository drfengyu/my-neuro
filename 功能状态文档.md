## 🔧 快速启用指南

### 启用RAG知识库

1. 启动RAG服务：
   ```bash
   cd full-hub
   python run_rag.py
   ```

2. 配置已经是 `enabled: true`，服务启动后自动生效

### 启用本地TTS

1. 修改 `live-2d/config.json`：
   ```json
   "tts": {
     "enabled": true,
     "url": "http://127.0.0.1:5000"
   },
   "cloud": {
     "aliyun_tts": {
       "enabled": false
     }
   }
   ```

2. 启动TTS服务：
   ```bash
   cd full-hub/tts-hub/GPT-SoVITS-Bundle
   python api_v2.py
   ```

### 启用视觉能力

1. 配置视觉模型API：
   ```json
   "vision": {
     "enabled": true,
     "auto_screenshot": true,
     "use_vision_model": true,
     "vision_model": {
       "api_key": "你的密钥",
       "api_url": "https://api.example.com/v1",
       "model": "gpt-4-vision-preview"
     }
   }
   ```

### 启用MCP工具

1. 编辑 `live-2d/mcp/mcp_config.json` 添加工具
2. 修改 `config.json`：
   ```json
   "mcp": {
     "enabled": true,
     "auto_start_servers": true
   }
   ```

## ⚠️ 注意事项

1. **本地TTS vs 云端TTS**:
   - 本地：需GPU，延迟低，可自定义
   - 云端：无需GPU，稳定，音质好

2. **RAG服务**:
   - 首次运行需下载模型
   - 占用约2GB内存

3. **视觉功能**:
   - 需要支持视觉的多模态模型
   - 会增加token消耗

4. **MCP工具**:
   - 需要Node.js环境
   - 某些工具需要额外权限
