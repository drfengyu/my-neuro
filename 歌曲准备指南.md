# 🎵 AI 唱歌功能 - 歌曲准备工具

这个目录包含下载和准备歌曲的工具。

## 📋 步骤总览

1. **下载歌曲** → 使用 `download-music.js`
2. **分离音频** → 使用第三方工具（见下方）
3. **移动文件** → 放到 `song-library/output/`
4. **播放测试** → 使用音乐控制面板

---

## 🔽 步骤 1：下载歌曲

### 安装依赖（首次使用）

```bash
npm install axios
```

### 搜索歌曲


## 🎛️ 步骤 2：分离音频（伴奏 + 人声）

下载的是完整音频，需要分离成伴奏和人声。

### 方法 1：使用 UVR（推荐，免费开源）

**Ultimate Vocal Remover (UVR)**

1. **下载安装**：
   - 官网：https://github.com/Anjok07/ultimatevocalremovergui/releases
   - 下载 Windows 版本（.exe 安装包）

2. **使用步骤**：
   - 打开 UVR
   - 点击 "Select Input" 选择下载的 MP3 文件
   - 选择模型：推荐 `UVR-MDX-NET-Inst_HQ_3`（高质量）
   - 点击 "Start Processing"
   - 等待处理完成

3. **输出文件**：
   ```
   song-library/downloads/
   ├── 告白气球-周杰伦.mp3          (原文件)
   ├── 告白气球-周杰伦_(Vocals).wav  (人声)
   └── 告白气球-周杰伦_(Instrumental).wav  (伴奏)
   ```

4. **重命名文件**（重要）：
   ```
   告白气球_(Vocals).wav     →  告白气球-Vocal.mp3
   告白气球_(Instrumental).wav  →  告白气球-Acc.mp3
   ```

---

### 方法 2：使用 Spleeter（命令行）

**安装**：
```bash
pip install spleeter
```

**使用**：
```bash
# 分离为人声和伴奏
spleeter separate -p spleeter:2stems -o song-library/output song-library/downloads/告白气球-周杰伦.mp3
```

**输出**：
```
song-library/output/告白气球-周杰伦/
├── vocals.wav      (人声)
└── accompaniment.wav  (伴奏)
```

**重命名**：
```bash
mv vocals.wav 告白气球-Vocal.mp3
mv accompaniment.wav 告白气球-Acc.mp3
```

   - 自动分离
   - 下载伴奏和人声

## 📦 步骤 3：整理文件

将分离后的文件放到正确位置：

```
my-neuro/
└── song-library/
    └── output/
        ├── 告白气球-Acc.mp3       ← 伴奏
        ├── 告白气球-Vocal.mp3     ← 人声
        ├── 告白气球.json          ← 元数据（自动生成）
        ├── 晴天-Acc.mp3
        ├── 晴天-Vocal.mp3
        └── ...
```

**文件命名规则**（必须遵守）：
- 伴奏：`歌曲名-Acc.mp3`
- 人声：`歌曲名-Vocal.mp3`
- 元数据：`歌曲名.json`（可选）

## 🚀 快速开始示例

### 示例：下载《告白气球》

```bash
# 1. 搜索歌曲
node download-music.js 告白气球

# 输出：
# 1. 告白气球 - 周杰伦 | ID: 413812448

# 2. 下载
node download-music.js 413812448

# 输出：
# ✅ 下载完成: song-library/downloads/告白气球-周杰伦.mp3
# 📝 元数据已保存: song-library/downloads/告白气球-周杰伦.json
```

### 3. 使用 UVR 分离音频

1. 打开 UVR
2. 选择 `song-library/downloads/告白气球-周杰伦.mp3`
3. 点击 "Start Processing"
4. 等待完成

### 4. 重命名文件

```bash
# 假设 UVR 输出在同一目录
cd song-library/downloads

# 重命名
ren "告白气球-周杰伦_(Vocals).wav" "告白气球-Vocal.mp3"
ren "告白气球-周杰伦_(Instrumental).wav" "告白气球-Acc.mp3"
```

### 5. 移动到输出目录

```bash
# 创建输出目录（如果不存在）
mkdir song-library\output

# 移动文件
move 告白气球-Acc.mp3 ..\output\
move 告白气球-Vocal.mp3 ..\output\
move 告白气球-周杰伦.json ..\output\告白气球.json
```

### 6. 播放测试

打开 `music-control-panel.html`，输入 `告白气球-Acc.mp3`，点击播放！

## ⚠️ 注意事项

1. **版权**：下载的歌曲仅供个人学习使用，请勿用于商业用途
2. **音质**：脚本默认下载"较高音质"，可在代码中修改 `level` 参数
3. **分离质量**：
   - UVR 质量最好，但处理较慢
   - Spleeter 速度快，质量中等
   - 在线工具方便，但有额度限制
4. **文件格式**：支持 .mp3, .wav, .m4a, .ogg

## 🔧 故障排查

### 问题 1：下载失败 "获取链接失败"
**原因**：歌曲可能有版权保护或需要 VIP  
**解决**：尝试搜索其他版本或歌曲

### 问题 2：分离后音质差
**原因**：原始音频质量低或模型不适合  
**解决**：
- 使用 UVR 的高质量模型
- 下载更高音质的音频

### 问题 3：播放时只有伴奏没有人声
**原因**：文件命名不符合规范  
**解决**：确保文件名为 `歌曲名-Acc.mp3` 和 `歌曲名-Vocal.mp3`

### 问题 4：API 连接失败
**原因**：API 服务器不可用或网络问题  
**解决**：
- 检查网络连接
- 尝试使用代理
- 更换其他网易云 API
